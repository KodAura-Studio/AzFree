# AzFree

Azərbaycanın ən etibarlı rəqəmsal resurs kataloqu. Oyunlar, filmlər və proqram təminatı üçün təhlükəsiz qapı.

## Features
- **PWA Support**: Installable on mobile and desktop
- **Global Search**: High-performance fuzzy search using Fuse.js
- **Premium Design**: Modern, responsive UI with Tailwind CSS
- **Dark Mode**: Fully synchronized theme engine
- **Optimized Performance**: Near-instant page loads

## Tech Stack
- HTML5 / CSS3
- JavaScript (ES6+)
- Tailwind CSS
- Fuse.js
- Service Workers (PWA)
